# 🥊 10秒ジャブ連打チャレンジ (Hyakuretsu Jab Challenge)

AIがあなたのパンチを判定！Webカメラを使って遊ぶ、新感覚フィットネス・アーケードゲーム。
10秒間でどれだけ連打できるか？迫りくるボスを撃破せよ！

## 🎮 ゲーム概要
PCのWebカメラに向かってジャブ（パンチ）を繰り出すことで、画面内のキャラクターが攻撃します。
全3ラウンドのボスバトル形式で、ラウンドが進むごとに敵は強力（高HP）になります。
運動不足解消、ストレス発散、そして限界への挑戦に！

## ✨ 特徴
- **AI姿勢推定**: MediaPipe PoseLandmarkerを使用し、ブラウザだけで高精度なモーション認識を実現。特別な機材は不要です。
- **ボスバトルモード**: HPゲージを持つボス敵と対戦。パンチのフォームを検知してショットを自動発射する爽快仕様。
- **ラウンド制 & インターバル**: 3ラウンド構成。ラウンド間には30秒の休憩（インターバル）があり、フィットネスとしてもリズムよく楽しめます。
- **レトロフューチャー**: サイバーパンクなデザインと8bit風の自動生成効果音。

## 🕹️ 遊び方

### 必要なもの
- Webカメラ付きのPC
- モダンブラウザ（Chrome推奨）
- 上半身が動かせる程度のスペース（1〜2m程度）

### 手順
1. **カメラ許可**: 画面右の「カメラ許可」ボタンを押し、ブラウザのカメラ使用を許可してください。
2. **スタンバイ**: カメラに上半身（特に肩と肘）が映るように立ちます。準備ができるとステータスが緑色になります。
3. **スタート**: 「10秒スタート」ボタンまたはスペースキーを押して開始！
4. **バトル (Boss Battle)**:
   - 画面に向かってリズムよくジャブを打ちます（肘をしっかり伸ばすと判定されやすいです）。
   - パンチが検知されるとエネルギー弾が発射され、ボスにダメージを与えます。
   - 制限時間内にボスのHPを削りきればラウンドクリアです。
5. **インターバル**: 
   - ラウンドクリア後は30秒の休憩タイムがあります。深呼吸して次のラウンドに備えましょう。
   - 全3ラウンドをクリアするとゲームクリアです。

### キーボード操作（カメラなし/テスト用）
- **左パンチ**: `A` キー または `←` (左矢印)
- **右パンチ**: `L` キー または `→` (右矢印)
- **スタート**: `Space` キー

## 🛠️ 技術スタック
- **Frontend**: HTML5, CSS3, Vanilla JavaScript (Single file component)
- **AI/ML**: MediaPipe Tasks Vision (PoseLandmarker)
- **Audio**: Web Audio API (Real-time synthesis)
- **Deployment**: GitHub Pages (Ready)

外部アセット（画像ファイルや音声ファイル）は一切使用しておらず、すべてコード内で生成またはCDNから読み込んでいます。

